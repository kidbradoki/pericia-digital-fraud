# 🛡️ App de Perícia Digital e Anti-Fraude

Este projeto é uma ferramenta de auxílio à investigação de fraudes financeiras e golpes digitais. Desenvolvido para facilitar a análise técnica de comprovantes e identificação de padrões suspeitos.

## 🚀 Funcionalidades
- **Análise de Região Fiscal:** Identifica a origem do CPF com base na Receita Federal.
- **Detecção de Instituições de Risco:** Alerta automático para bancos digitais e IPs comumente usados em contas de passagem.
- **Análise de Chaves PIX:** Identificação de padrões de chaves aleatórias (EVP).

## 🛠️ Tecnologias
- **Backend:** Python com Framework Flask.
- **Frontend:** HTML5 e CSS3 (Interface responsiva).
- **Segurança:** Separação de camadas para proteção de lógica de negócio.
